#####Bot Commands#####

######IP Calculations######

***binary***

    <user> 1010
    <bot> {"dec":10,"hex":"a"}

***hex***

    <user> 0xFF
    <bot> {"dec":255,"bin":"11111111"}
    

######Quiz######

***View available certs***

    <user> certs
    <bot> icnd1, icnd2, ...
    

***Count cert database entries***
Syntax: count <cert name>

Example:
    
    <user> count icnd1
    <bot> 354
